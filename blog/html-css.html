Best Practices When Using Classes and IDs

Class and id are both very commonly used selectors for styling and building the layout of an html document. These selectors are used in CSS to target and style one or more elements on a web page.

An id attribute is a unique identifier used for a specific html element. Each element can have only one id and each page can have only one element with that specific id.

A class selector is not unique and can be applied to identify multiple elements.
